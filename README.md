# 📌 Collaborative Code Review Platform

## 📖 Description

A Node.js and TypeScript backend for reviewing and managing code with authentication and PostgreSQL.

---

## 🚀 Features

* User authentication (JWT)
* Password hashing (bcrypt)
* REST API with Express
* PostgreSQL database

---

## 🛠️ Tech Stack

Node.js • TypeScript • Express • PostgreSQL

---

## ⚙️ Setup

```bash
git clone https://github.com/Mluleki23/Collaborative-Code-Review-Platform.git
cd Collaborative-Code-Review-Platform
npm install
```

Create `.env`:

```
PORT=5000
DATABASE_URL=your_database_url
JWT_SECRET=your_secret
```

---

## ▶️ Run

```bash
npm run dev
```
